# ArmorCRA — Scripts de réponse aux objections

## "C'est légal votre truc ?"
"C'est de l'OSINT (Open Source Intelligence) passif et légal.
ArmorCRA utilise uniquement des vecteurs de données publics que n'importe quel
acteur malveillant ou auditeur de l'UE peut consulter.
Notre but est de prévenir l'entreprise avant que l'UE ne le fasse via une amende CRA."

## "D'où viennent vos données ?"
"Nos sources sont 100% publiques : dépôts GitHub publics, headers HTTP,
certificats SSL, ports ouverts, et rapports ANSSI.
Aucune intrusion, aucun accès non autorisé. Tout ce qu'ArmorCRA détecte
est visible par n'importe quel scanner automatisé — y compris ceux des hackers."

## "On est déjà conformes"
"Parfait. Dans ce cas votre score ArmorCRA devrait être au-dessus de 75/100.
Voulez-vous qu'on le vérifie ensemble ? Si c'est le cas, on vous certifie
'CRA Ready' et ça peut devenir un argument commercial pour vos clients."

## "On n'a pas de budget pour ça"
"Je comprends. Pour information, une amende CRA représente 2,5% de votre CA mondial.
Sur 2M€ de CA, c'est 50 000€ minimum. La mise en conformité coûte généralement
3 000€ à 8 000€. Le ROI est immédiat."

## "On verra ça l'année prochaine"
"Le CRA entre en vigueur en décembre 2027, mais les audits de conformité commencent
dès 2026. Les assureurs cyber ont déjà commencé à refuser les renouvellements
pour les entreprises non conformes. Attendre coûte plus cher qu'agir maintenant."
